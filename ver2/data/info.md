# Описание данных

Папка `ver2/data` содержит справочные данные о суточных нормах потребления витаминов и минералов.

## Файлы

| Файл | Описание |
|------|----------|
| `vitamins.json` | Нормы потребления витаминов по возрастным группам |
| `minerals.json` | Нормы потребления минералов по возрастным группам |

---

## Источники данных

Данные взяты из официальных таблиц Dietary Reference Intakes (DRI) Министерства сельского хозяйства США (USDA):

- **Основной источник**: [DRI Tables — National Academies](https://www.ncbi.nlm.nih.gov/books/NBK56056/)
- **Дополнительный источник**: [DRI Values — NCBI](https://www.ncbi.nlm.nih.gov/books/NBK222309/)
- **База данных продуктов питания**: [FoodData Central — USDA](https://fdc.nal.usda.gov/)

### Типы нормативных значений

| Обозначение | Полное название | Описание |
|-------------|-----------------|----------|
| **RDA** | Recommended Dietary Allowance | Рекомендуемая суточная норма — средний дневной уровень потребления, достаточный для удовлетворения потребностей 97–98% здоровых людей данной группы |
| **AI** | Adequate Intake | Достаточное потребление — устанавливается при нехватке данных для RDA; предполагается, что обеспечивает достаточное питание |
| **UL** | Tolerable Upper Intake Level | Верхний допустимый уровень — максимальный суточный уровень, не представляющий риска для здоровья |

## Структура JSON

Оба файла имеют одинаковую структуру верхнего уровня:

```json
{
  "metadata": { ... },
  "vitamins": [ ... ]   // или "minerals": [ ... ]
}
```

### Объект `metadata`

```json
{
  "source":      "Название источника данных",
  "source_url":  "Ссылка на первичный источник",
  "source_url_2": "Ссылка на дополнительный источник",
  "description": "Краткое описание содержимого файла"
}
```

### Элемент массива `vitamins` / `minerals`

```json
{
  "id":       "уникальный идентификатор (например: vit_d, iron)",
  "name":     "Название на русском языке",
  "name_en":  "Название на английском языке",
  "unit":     "Единица измерения на русском (мг, мкг)",
  "unit_en":  "Единица измерения на английском (mg, mcg)",
  "groups":   [ ... ]
}
```

### Элемент массива `groups`

Каждый элемент описывает норму для одной возрастной/физиологической группы:

```json
{
  "life_stage":    "Код группы на английском (например: 19-30 years)",
  "life_stage_ru": "Название группы на русском (например: 19-30 лет)",
  "rda":  15,    // число или null, если не определено
  "ai":   null,  // число или null, если не определено
  "ul":   100    // число или null, если не определено
}
```

> Примечание: для каждой группы задаётся либо `rda`, либо `ai` (но не оба одновременно). Если значение не определено, поле равно `null`.

### Возрастные группы

| Код (`life_stage`) | Описание |
|--------------------|----------|
| `0-6 months` | Младенцы 0–6 месяцев |
| `6-12 months` | Младенцы 6–12 месяцев |
| `1-3 years` | Дети 1–3 года |
| `4-8 years` | Дети 4–8 лет |
| `9-13 years` | Дети 9–13 лет |
| `14-18 years` | Подростки 14–18 лет |
| `19-30 years` | Взрослые 19–30 лет |
| `31-50 years` | Взрослые 31–50 лет |
| `51-70 years` | Взрослые 51–70 лет |
| `70+ years` | Пожилые старше 70 лет |
| `pregnancy 14-18` | Беременные 14–18 лет |
| `pregnancy 19-30` | Беременные 19–30 лет |
| `pregnancy 31-50` | Беременные 31–50 лет |
| `lactation 14-18` | Кормящие 14–18 лет |
| `lactation 19-30` | Кормящие 19–30 лет |
| `lactation 31-50` | Кормящие 31–50 лет |
