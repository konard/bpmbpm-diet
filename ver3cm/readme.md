# Трекер питания ver3cm

Браузерное приложение для учёта калорий, КБЖУ, витаминов и минералов. Создано по образцу [cronometer.com](https://cronometer.com).

## Возможности

- **Дневник питания** — 4 приёма пищи с поиском продуктов и расчётом КБЖУ
- **Дашборд** — итоги дня, недельный график, прогресс витаминов/минералов
- **Профиль** — рост (см), вес (кг), ИМТ, BMR, TDEE, цели
- **Упражнения** — запись активности и сожжённых калорий
- **Русский / Английский** язык интерфейса

## Пользователи

Кнопка **«Пользователь»** в шапке:
- **Загрузить данные пользователя** — выбор Test1, Test2, Test3
- **Сохранить данные пользователя** — скачивание JSON
- **Загрузить тестовые данные** — быстрый старт с Test1

Файлы пользователей: `User/Test1.json`, `User/Test2.json`, `User/Test3.json`

Добавление пользователей: отредактируйте `user.js`

## Структура файлов

```
ver3cm/
├── index.html, style.css, script.js, user.js
├── data/    — foods.json, vitamins.json, minerals.json, data_info.md
├── User/    — Test1.json, Test2.json, Test3.json
├── doc/     — architecture.md, data_format.md, features.md
└── help/    — help.md, diary.md, profile.md, nutrients.md
```

## Запуск

Откройте `index.html` через веб-сервер (GitHub Pages, или `python3 -m http.server` локально).

> Прямое открытие через `file://` может вызвать ошибку загрузки данных в Chrome.
