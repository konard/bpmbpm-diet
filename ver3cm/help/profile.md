# Профиль и цели

## Для чего нужен профиль

Данные профиля используются для расчёта:
- **ИМТ** (индекс массы тела)
- **BMR** (базовый обмен веществ)
- **TDEE** (суточные энергозатраты с учётом активности)
- Целевых норм потребления белков, жиров и углеводов

---

## Поля профиля

| Поле | Описание |
|------|----------|
| **Пол** | Мужской или женский — влияет на формулу BMR |
| **Возраст** | В полных годах (13–80 лет) |
| **Рост** | В сантиметрах |
| **Вес** | В килограммах |
| **Физическая активность** | Коэффициент для расчёта TDEE |
| **Цель** | Снижение / поддержание / набор веса |
| **Доля белков, %** | Процент калорий из белка |
| **Доля жиров, %** | Процент калорий из жира |
| **Доля углеводов, %** | Процент калорий из углеводов |

> Сумма долей белков + жиров + углеводов нормализуется до 100%.

---

## Уровни физической активности

| Коэффициент | Описание |
|-------------|----------|
| 1,2 | Минимальная (сидячая работа) |
| 1,375 | Слабая (1–3 тренировки в неделю) |
| 1,55 | Умеренная (3–4 тренировки в неделю) |
| 1,7 | Высокая (5–7 дней в неделю) |
| 1,9 | Экстремальная (ежедневно + физический труд) |

---

## Расчёт ИМТ

> **ИМТ = вес (кг) / рост² (м)**

| ИМТ | Категория |
|-----|-----------|
| < 18,5 | Дефицит массы тела |
| 18,5 – 25,0 | Норма |
| 25,0 – 30,0 | Избыток |
| > 30,0 | Ожирение |

---

## Формула Миффлина-Сан Жеора (BMR)

- **Мужчины:** 10 × вес + 6,25 × рост − 5 × возраст + 5
- **Женщины:** 10 × вес + 6,25 × рост − 5 × возраст − 161

**TDEE = BMR × коэффициент активности**

---

## Расчёт цели по калориям

| Цель | Формула |
|------|---------|
| Снижение веса | TDEE × 0,8 (дефицит 20%) |
| Поддержание | TDEE |
| Набор массы | TDEE × 1,15 (профицит 15%) |
