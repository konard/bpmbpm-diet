# Общая справка — Трекер питания

## О программе

**Трекер питания** — браузерное приложение для учёта калорий, белков, жиров и углеводов (КБЖУ), а также витаминов и минералов. Работает без регистрации и подключения к интернету (после первой загрузки).

---

## Основные возможности

| Раздел | Описание |
|--------|----------|
| **Дневник** | Записывайте, что вы съели на завтрак, обед, ужин и в качестве перекуса |
| **Дашборд** | Видите итоговую статистику за день и недельный график калорий |
| **Профиль** | Укажите рост, вес, возраст и цель — программа рассчитает ИМТ и суточную норму калорий |
| **Упражнения** | Записывайте физическую активность и сожжённые калории |
| **Справка** | Эта страница и другие справочные материалы |

---

## Навигация

В верхней части экрана находится панель вкладок:
- Нажмите на название вкладки, чтобы перейти в соответствующий раздел
- Текущий раздел выделен синей чертой

---

## Кнопка «Пользователь»

В правом верхнем углу находится выпадающее меню **«Пользователь»** с тремя функциями:

1. **Загрузить данные пользователя** — выбрать один из сохранённых профилей (Test1, Test2, Test3)
2. **Сохранить данные пользователя** — скачать текущие данные на ваш компьютер в формате JSON
3. **Загрузить тестовые данные** — быстро загрузить пример заполненного дневника (Test1)

---

## Переключение языка

Нажмите кнопку **EN** (или **RU**) в правом верхнем углу для переключения между русским и английским языком интерфейса.

---

## Добавление собственного пользователя

Откройте файл `user.js` в текстовом редакторе и добавьте новую запись в массив `SPISOK_POLZOVATELEY`:

```js
{
    id: 'MyUser',
    nazvanie: 'Мой пользователь',
    fajl: 'User/MyUser.json'
}
```

Затем создайте файл `User/MyUser.json` в том же формате, что и `User/Test1.json`.

---

## Технические сведения

- Приложение работает полностью в браузере (чистый JavaScript, без серверной части)
- Данные хранятся только в памяти браузера — при перезагрузке страницы они сбрасываются
- Используйте **«Сохранить данные пользователя»** для постоянного хранения ваших данных
- Совместимо со всеми современными браузерами (Chrome, Firefox, Safari, Edge)
