# Форматы данных

## Формат файла пользователя (User/*.json)

```json
{
  "version": "1.0",
  "profil": {
    "pol":              "m",      // "m" — мужской, "f" — женский
    "vozrast":          35,       // возраст в годах (целое число)
    "rost":             178,      // рост в сантиметрах
    "ves":              82,       // вес в килограммах
    "aktivnost":        1.55,     // коэффициент активности (1.2 / 1.375 / 1.55 / 1.7 / 1.9)
    "cel":              "lose",   // "lose" / "maintain" / "gain"
    "belokProcent":     30,       // доля белков в % от калорий
    "zhirProcent":      30,       // доля жиров в %
    "uglevodyProcent":  40        // доля углеводов в %
  },
  "dnevnik": {
    "2026-03-25": {               // ключ: дата YYYY-MM-DD
      "zavtrak":  [...],          // завтрак
      "obed":     [...],          // обед
      "uzhin":    [...],          // ужин
      "perekus":  [...]           // перекус
    }
  },
  "uprazhneniya": {
    "2026-03-25": [               // ключ: дата YYYY-MM-DD
      {
        "nazvanie":         "Бег",  // название упражнения
        "minuty":           30,     // длительность в минутах
        "kaloriiSozhzheno": 280     // сожжённые калории
      }
    ]
  }
}
```

## Формат базы продуктов (data/foods.json)

```json
{
  "metadata": {
    "source":      "USDA FoodData Central",
    "source_url":  "https://fdc.nal.usda.gov/",
    "description": "База продуктов питания — КБЖУ и основные микроэлементы на 100 г"
  },
  "products": [
    {
      "id":          "chicken_breast_boiled",       // уникальный идентификатор
      "name":        "Куриная грудка (отварная)",  // название на русском
      "name_en":     "Chicken breast (boiled)",    // название на английском
      "category":    "Мясо",                       // категория на русском
      "category_en": "Meat",                       // категория на английском
      "per100g": {
        "kcal":     165,     // калории
        "protein":  31.0,    // белки, г
        "fat":      3.6,     // жиры, г
        "carbs":    0.0,     // углеводы, г
        "fiber":    0.0,     // клетчатка, г
        "vit_d":    0.1,     // витамин D, мкг
        "vit_c":    0,       // витамин C, мг
        "vit_a":    6,       // витамин A, мкг RAE
        "vit_b12":  0.3,     // витамин B12, мкг
        "iron":     1.0,     // железо, мг
        "calcium":  15,      // кальций, мг
        "magnesium":29,      // магний, мг
        "potassium":256,     // калий, мг
        "sodium":   74,      // натрий, мг
        "zinc":     1.0      // цинк, мг
      }
    }
  ]
}
```

## Формат файлов витаминов/минералов (data/vitamins.json, data/minerals.json)

Подробное описание см. в `data/data_info.md`.

Краткая схема:
```json
{
  "metadata": { ... },
  "vitamins": [
    {
      "id":    "vit_d",
      "name":  "Витамин D",
      "unit":  "мкг",
      "groups": [
        {
          "life_stage": "19-30 years",
          "rda":  15,
          "ai":   null,
          "ul":   100
        }
      ]
    }
  ]
}
```

### Коды возрастных групп

| Код | Описание |
|-----|----------|
| `0-6 months` | Младенцы 0–6 мес |
| `1-3 years` | Дети 1–3 года |
| `4-8 years` | Дети 4–8 лет |
| `9-13 years` | Дети 9–13 лет |
| `14-18 years` | Подростки 14–18 лет |
| `14-18 years f` | Девочки 14–18 лет |
| `19-30 years` | Взрослые 19–30 лет |
| `19-30 years f` | Женщины 19–30 лет |
| `31-50 years` | Взрослые 31–50 лет |
| `31-50 years f` | Женщины 31–50 лет |
| `51-70 years` | Взрослые 51–70 лет |
| `70+ years` | Старше 70 лет |
