# Описание справочных данных (ver3cm/data/)

Папка `ver3cm/data` содержит справочные данные о продуктах питания и суточных нормах потребления питательных веществ.

---

## Файлы

| Файл | Описание |
|------|----------|
| `foods.json` | База продуктов питания: КБЖУ и основные микроэлементы на 100 г |
| `vitamins.json` | Нормы потребления витаминов (RDA/AI/UL) по возрастным группам |
| `minerals.json` | Нормы потребления минералов (RDA/AI/UL) по возрастным группам |

---

## foods.json

### Источник

**USDA FoodData Central**
- URL: https://fdc.nal.usda.gov/
- Описание: официальная база данных состава продуктов питания Министерства сельского хозяйства США

### Структура

```json
{
  "metadata": { "source", "source_url", "description" },
  "products": [
    {
      "id":          "уникальный_идентификатор",
      "name":        "Название на русском",
      "name_en":     "English name",
      "category":    "Категория на русском",
      "category_en": "Category in English",
      "per100g": {
        "kcal":      число,   // калории на 100 г
        "protein":   число,   // белки, г
        "fat":       число,   // жиры, г
        "carbs":     число,   // углеводы, г
        "fiber":     число,   // пищевые волокна, г
        "vit_d":     число,   // витамин D, мкг
        "vit_c":     число,   // витамин C, мг
        "vit_a":     число,   // витамин A, мкг RAE
        "vit_b12":   число,   // витамин B12, мкг
        "iron":      число,   // железо, мг
        "calcium":   число,   // кальций, мг
        "magnesium": число,   // магний, мг
        "potassium": число,   // калий, мг
        "sodium":    число,   // натрий, мг
        "zinc":      число    // цинк, мг
      }
    }
  ]
}
```

## vitamins.json

### Источник

**USDA Dietary Reference Intakes (DRI)**
- Основной URL: https://www.ncbi.nlm.nih.gov/books/NBK56056/
- Дополнительный URL: https://www.ncbi.nlm.nih.gov/books/NBK222309/
- Описание: официальные таблицы рекомендуемых норм потребления питательных веществ

### Структура

```json
{
  "metadata": { "source", "source_url", "source_url_2", "description" },
  "vitamins": [
    {
      "id":         "vit_d",
      "name":       "Витамин D",
      "name_en":    "Vitamin D",
      "unit":       "мкг",
      "unit_en":    "mcg",
      "groups": [
        {
          "life_stage":    "19-30 years",
          "life_stage_ru": "19-30 лет",
          "rda":  15,    // число или null
          "ai":   null,  // число или null
          "ul":   100    // число или null
        }
      ]
    }
  ]
}
```

| ID | Название | Единица |

| `vit_d` | Витамин D | мкг |

### Источник

### Структура

Аналогична vitamins.json, но ключ верхнего уровня — `"minerals"` вместо `"vitamins"`.
